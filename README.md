# AppointmentBot
Call to schedule an appointment with your local doctor

# Setup
Setup instructions can be found here: https://docs.vocode.dev/telephony#self-hosted

Make sure you have Following in your .env file (use as template): 

BASE_URL=
DEEPGRAM_API_KEY=
OPENAI_API_KEY=
AZURE_SPEECH_KEY=
AZURE_SPEECH_REGION=
TWILIO_ACCOUNT_SID=
TWILIO_AUTH_TOKEN=
NGROK_AUTH_TOKEN=
